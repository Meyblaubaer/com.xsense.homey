X-Sense for Homey

Integrate X-Sense smoke detectors, CO alarms, and environmental sensors into Homey.

Supported Devices:
- Smoke & CO Detectors (e.g., XS01-WX, SC07-WX)
- WiFi Thermometer & Hygrometers (e.g., STH51)
- Water Leak Detectors (e.g., SWS51)

Features:
- Real-time alarms for smoke, CO, and water leaks
- Battery status monitoring
- Temperature and humidity readings
- Flow triggers for automation

Usage:
IMPORTANT: Use a dedicated X-Sense account for Homey (via Family Key sharing) to avoid session conflicts.

1. Install this app.
2. Add a new device in Homey.
3. Log in with your X-Sense account credentials.
